PopupMenu.PopupMenuFactory

PopupMenu.PopupMenuFactory — A class to build a cinnamon menu using some abstract menu items.

Object Hierarchy

    Object
    ╰── PopupMenu.PopupMenuFactory

Description

This class can build a cinnamon menu, using the instances of a heir of the PopupMenuAbstractItem class. Please see the description of the PopupMenuAbstractItem class to more details. To initialize the construction you need to provide the root instance of your abstract menu items.
