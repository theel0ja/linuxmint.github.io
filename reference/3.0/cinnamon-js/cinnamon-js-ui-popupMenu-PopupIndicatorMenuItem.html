PopupMenu.PopupIndicatorMenuItem — A menu item with text, ornaments and accel.

Object Hierarchy

Description

This is a popup menu item displaying an text, a accel, and a ornament. The ornament is displayed to the left of the text and the accel will be displayed at the end of the item. The default ornament is an icon, but can be replace for a check button, a radio button or empty.
